## Why Tracelet?

Tracelet flips the traditional location tracking model. Instead of relying purely on high-frequency, battery-draining GPS polls, we use intelligent **Motion-detection algorithms**, **activity recognition**, and **Adaptive Sampling** to track devices seamlessly in the background with near-zero battery drain when stationary.

<Callout type="info" emoji="🔋">
  **Battery Budget Engine:** Automatically tune `distanceFilter` and `desiredAccuracy` on the fly to stay within a configurable battery drain target (e.g. `2.0%` per hour).
</Callout>

## Federated Architecture

Tracelet uses a **federated plugin architecture** with 5 packages, ensuring modularity and performance:

| Package | Description |
|---|---|
| `tracelet` | App-facing Dart API — the only package you depend on |
| `tracelet_doctor` | Advanced in-app diagnostic dashboard and health analysis overlay UI |
| `tracelet_platform_interface` | Abstract platform interface + Pigeon definitions |
| `tracelet_android` | Kotlin Android implementation (standalone SDK available) |
| `tracelet_ios` | Swift iOS implementation (standalone SDK available) |
