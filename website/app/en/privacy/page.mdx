# Privacy Policy

**Effective Date: June 12, 2026**

Tracelet ("we," "our," or "us") is an open-source background geolocation and geofencing engine for Flutter. 

## Data Collection
Tracelet operates entirely locally on your users' devices. Our core open-source package **does not** collect, transmit, or store any personal data, location data, or telemetrics to Tracelet's own servers.

Any location data captured by Tracelet is stored locally in an SQLite database on the device, or transmitted directly to your own configured backend via `tracelet_sync`.

## User Responsibility
Because Tracelet empowers developers to capture highly sensitive background location data, it is the sole responsibility of the developer integrating Tracelet to:
1. Obtain the necessary explicit location and motion permissions from end-users.
2. Disclose the collection and usage of location data in their application's own Privacy Policy.
3. Secure the backend servers where Tracelet's data is synced.

We strongly encourage all developers using Tracelet to adhere to platform guidelines set by Google Play and the Apple App Store regarding location tracking.
