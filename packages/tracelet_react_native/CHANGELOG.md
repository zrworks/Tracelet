# Changelog

## 1.0.0-alpha.1

- **FEAT**: Initial React Native binding for Tracelet (`@ikolvi/tracelet`).
- **FEAT**: Full TypeScript facade mirroring the Dart `Tracelet` API 1:1 — tracking lifecycle, on-demand location, circular + polygon geofencing, SQLite persistence, HTTP sync, telematics, crash & fall detection, and enterprise audit/privacy features.
- **FEAT**: Android module wired to the published `com.ikolvi:tracelet-sdk` native SDK.
- **FEAT**: iOS module wired to the published `TraceletSDK` native SDK.
- **FEAT**: React hooks (`useLocation`, `useTraceletState`, `useGeofences`).
