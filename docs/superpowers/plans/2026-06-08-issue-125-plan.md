# Issue 125 Sync Timeout Abort Fix Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Fix Issue #125 by ensuring that if a custom sync body builder times out or errors, the native SDKs completely abort the sync attempt rather than posting error JSON payloads or default telemetry.

**Architecture:** We will modify `TraceletIosPlugin.swift` to return `nil` on error/timeout instead of literal JSON error strings. In both Android and iOS sync provider layers, we will check if the dart interceptor is present. If it returns `nil` (indicating a failure to build the custom body), we immediately return `0` (abort) instead of falling back to the default `syncBatchBlocking`. The example app will include a button that intentionally triggers a timeout to visually prove no request is fired.

**Tech Stack:** Kotlin, Swift, Dart, Flutter Integration Tests

```dart
import 'dart:async';
import 'package:flutter_test/flutter_test.dart';
import 'package:integration_test/integration_test.dart';
import 'package:tracelet/tracelet.dart' as tl;

void main() {
  IntegrationTestWidgetsFlutterBinding.ensureInitialized();

  testWidgets('Issue 125: Custom sync body timeout aborts sync without fallback payload', (tester) async {
    await tl.Tracelet.stop();
    await tl.Tracelet.clearDatabase();

    tl.Tracelet.onHttpEvent((event) {
      if (event.status == 200) {
        fail('HTTP event should not succeed because sync should have aborted.');
      }
    });

    await tl.Tracelet.initialize(
      config: tl.TraceletConfig(
        http: tl.HttpConfig(
          url: 'http://localhost:8099/issue125',
          autoSync: true,
          autoSyncDelay: 1000,
        ),
      ),
    );

    // Intentionally hang the custom builder to trigger native DART_CALLBACK_TIMEOUT (10s)
    tl.Tracelet.onBuildSyncBody((locations) async {
      await Future.delayed(const Duration(seconds: 15));
      return '{"custom": true}';
    });

    await tl.Tracelet.insertMockLocation(lat: 5.0, lng: 5.0);

    // Wait for the native timeout (10s) + some buffer
    await tester.pumpAndSettle(const Duration(seconds: 12));

    // If we reach here without the `fail` in `onHttpEvent` triggering, 
    // it means the sync aborted cleanly and no error payload was sent.
    expect(true, isTrue);
  });
}
```
